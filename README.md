# Pringle-scope

Low-cost wireless telescope prototype: small mirror, CMOS sensor with microcontroller firmware, and 3D-printed mechanical parts—aimed at networked citizen science (e.g. stacking images from many small scopes).

**Status:** Early prototype / research (Human Computation Institute).

**Repository:** https://github.com/hcinst/pringle-scope

## Repository layout (target)

| Path | Contents |
|------|----------|
| `firmware/` | Arduino / MCU sketches and notes |
| `hardware/3d-print/` | STL/STEP exports and design revision notes |
| `docs/` | Design decisions, calibration, safety |
| `kit/` | BOM, sourcing, assembly checklists (as they exist) |

Folders will fill in as the project matures.
